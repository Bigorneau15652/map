Comment renseigner la colonne

Selon le type de colonne Grist
Colonne « Pièces jointes »
Utilisez le type de colonne Pièces jointes dans Grist et déposez-y vos fichiers directement (images ou PDF). Le widget les affiche automatiquement, dans l'ordre de dépôt.
Colonne « Texte » — plusieurs URLs (une par ligne)
https://upvdrive.univ-montp3.fr/s/rcaX7329CWkwodF
https://upvdrive.univ-montp3.fr/s/apbtGpLMs9gj4Kg
https://upvdrive.univ-montp3.fr/s/xYzAbcDef12345
Séparateurs acceptés : saut de ligne, virgule , ou point-virgule ;
Liens Nextcloud — formats acceptés
https://upvdrive.univ-montp3.fr/s/rcaX7329CWkwodF
https://upvdrive.univ-montp3.fr/s/rcaX7329CWkwodF/
https://upvdrive.univ-montp3.fr/s/rcaX7329CWkwodF/download
Le widget ajoute automatiquement /download aux liens de partage Nextcloud qui n'en ont pas.
Mix images et PDF
https://upvdrive.univ-montp3.fr/s/facade-nord
https://upvdrive.univ-montp3.fr/s/plan-rdc.pdf
https://upvdrive.univ-montp3.fr/s/photo-interieur
Les PDF sont détectés automatiquement (extension .pdf ou Content-Type). Les points indicateurs en bas sont rouges pour les PDF, blancs pour les images.

Conseil d'organisation

Pour un bâtiment avec plusieurs vues : saisissez une URL par ligne dans la colonne texte — façade nord, façade sud, plan RDC, plan R+1, etc. Le widget affiche la première image au chargement. Utilisez ← → pour parcourir. Lorsque vous changez de ligne (autre bâtiment), la visionneuse se remet automatiquement sur la première image.
