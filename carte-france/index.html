❓ Guide d'utilisation – Carte de France Grist

1. Tables à créer dans Grist

Ce widget utilise 3 tables dans votre document Grist :

TableRôle
DepartementsLes 101 départements (importer le CSV fourni)
CategoriesVos thèmes de coloration (ex: Zone montagne)
Departement_categoriesLes liens dept ↔ catégorie (géré par le widget)

2. Colonnes requises

Table Departements — importer depuis :
https://bigorneau15652.github.io/map/carte-france/departements.csv

  • code_dept — code INSEE (01, 75, 971…)
  • nom_dept — nom du département
  • visible — Bascule — si décoché, le département est grisé sur la carte

Table Categories

  • nom — ex: Zone montagne
  • couleur — nom FR (rouge, bleu…) ou HEX (#e74c3c)
  • active — Bascule — si coché, la catégorie est visible sur la carte

Table Departement_categories — laisser vide, le widget la remplit

  • dept — Référence → Departements
  • categorie — Référence → Categories
  • valeur — Numérique (optionnel, pour mode Choroplèthe)

3. Ajouter le widget dans Grist

  • Dans une page Grist : Ajouter un widget → Widget personnalisé
  • URL : https://bigorneau15652.github.io/map/carte-france/
  • Niveau d'accès : Accès complet au document
  • Pas besoin de lier à une table spécifique

4. Configurer les noms de tables (⚙️)

Si vos tables s'appellent différemment, cliquez ⚙️ en haut à droite du widget et entrez les noms exacts.

Pour trouver le nom interne d'une table Grist : Outils → Vue du code et repérez l'identifiant Python.

6. Workflow d'utilisation

  • Cochez active sur une catégorie → ses départements s'illuminent
  • Dans le widget, cliquez la pastille colorée d'un département pour l'ajouter/retirer de la catégorie
  • Décochez la case à gauche d'un département → il devient grisé sur la carte
  • Un département dans plusieurs catégories actives prend la couleur cumul (réglable via le sélecteur orange)
